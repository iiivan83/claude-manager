- **Таблицы заменены на списки**: примеры хороших имён файлов оформлены маркированным списком вместо markdown-таблицы. **Причина**: глобальное правило в `~/.claude/CLAUDE.md` запрещает таблицы — они ломаются при пересылке в Telegram.
- **Примеры адаптированы под контекст проекта**: вместо courier/salary/contract — session_manager/process_manager/claude_runner; вместо миграции — протокол stream-json и архитектура бота. **Причина**: примеры должны быть релевантны текущему проекту.
- **Добавлено разграничение с update-docs**: в description добавлено «НЕ используй для обновления существующих документов — для этого есть скилл update-docs». **Причина**: предотвращение конфликта триггеров между скиллами create-doc и update-docs.
