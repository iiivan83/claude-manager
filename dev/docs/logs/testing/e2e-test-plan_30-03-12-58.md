## CJM-04: Создание новой сессии

### CJM-04-01: Создание сессии через /new
- **Предусловие:** бот запущен, пользователь авторизован
- **Шаги:**
  1. Отправить команду `/new`
- **Ожидаемый результат:** бот отвечает подтверждением с номером сессии (например, «Создана новая сессия #N»)
- **Способ проверки:** LLM проверяет, что ответ содержит подтверждение создания сессии и номер

## CJM-02: Текстовое сообщение

### CJM-02-01: Обычное текстовое сообщение с ответом Claude
- **Предусловие:** пользователь подключён к сессии (после CJM-04-01)
- **Шаги:**
  1. Отправить сообщение: «Скажи одним предложением: что такое Python?»
- **Ожидаемый результат:** бот отвечает текстом от Claude с заголовком сессии и галочкой (формат: #N + галочка + ответ)
- **Способ проверки:** LLM проверяет наличие номера сессии с галочкой и осмысленного ответа про Python

### CJM-02-02: Сообщение без активной сессии
- **Предусловие:** пользователь НЕ подключён к сессии (после /all)
- **Шаги:**
  1. Отправить команду `/all` (переход в режим мониторинга)
  2. Отправить текстовое сообщение: «Привет»
- **Ожидаемый результат:** бот отвечает подсказкой — что нужно создать или выбрать сессию
- **Способ проверки:** LLM проверяет, что ответ содержит подсказку про /new или /sessions

## CJM-05: Просмотр списка сессий

### CJM-05-01: Список сессий через /sessions
- **Предусловие:** существует хотя бы одна сессия (после CJM-02-01)
- **Шаги:**
  1. Отправить команду `/sessions`
- **Ожидаемый результат:** бот отправляет список с номерами сессий и превью первых сообщений
- **Способ проверки:** LLM проверяет, что ответ содержит пронумерованные сессии в формате /N

## CJM-06: Переключение на сессию

### CJM-06-01: Переключение по номеру
- **Предусловие:** есть сессии в списке (после CJM-05-01)
- **Шаги:**
  1. Отправить команду с номером первой сессии из списка (например `/1`)
- **Ожидаемый результат:** бот подтверждает подключение к сессии с номером и превью
- **Способ проверки:** LLM проверяет наличие подтверждения подключения с номером сессии

## CJM-07: Мониторинг всех сессий

### CJM-07-01: Включение режима мониторинга через /all
- **Предусловие:** пользователь подключён к сессии
- **Шаги:**
  1. Отправить команду `/all`
- **Ожидаемый результат:** бот подтверждает переход в режим мониторинга всех сессий
- **Способ проверки:** LLM проверяет, что ответ содержит подтверждение режима мониторинга

## CJM-08: Остановка Claude

### CJM-08-01: Остановка через /stop без активной сессии
- **Предусловие:** пользователь в режиме /all (не подключён к сессии)
- **Шаги:**
  1. Убедиться, что пользователь в режиме /all
  2. Отправить команду `/stop`
- **Ожидаемый результат:** бот сообщает, что /stop работает только внутри сессии
- **Способ проверки:** LLM проверяет, что ответ содержит информацию о необходимости подключения к сессии
