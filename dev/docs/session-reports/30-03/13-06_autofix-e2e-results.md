## Детали по группам

### Группа 1: Claude CLI stdout buffering — ПОЧИНЕНО

- **Тесты:** CJM-02-01 (текстовое сообщение с ответом), CJM-03-01 (отправка фотографии)
- **Причина:** После отправки сообщения в stdin процесса Claude CLI бот не закрывал stdin. Claude CLI использует block buffering для stdout при работе через pipe — без EOF на stdin он ждёт ещё данные и не отдаёт ответ. Процесс зависал, бот получал «Процесс Claude завершился без события result» и уходил в цикл ретраев.
- **Исправление:** Добавлен `self.process.stdin.close()` в метод `send_message()` после записи сообщения (`claude_runner.py:98`). Claude CLI получает EOF и начинает обработку.
- **Попыток:** 1
- **Коммит:** f5cc1dc

## Коммиты
